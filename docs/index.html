Setup Required

Relic's web UI is a static frontend. It connects to a local API server running on your machine. You need to install a few things first:

1
Install Ollama

Download from ollama.com. This runs AI models locally on your GPU/CPU.

2
Pull a model

Run ollama pull glm4 in your terminal. Pick a model that fits your hardware from the Model selector.

3
Install Relic

Run pip install relic[web] or clone the repo and run pip install -e ".[web]"

4
Start the API

Run relic-web in your terminal. The API starts on 127.0.0.1:8746

5
Connect

Click Connect in the sidebar. Set your target and start scanning.

Nothing runs in the cloud. All AI inference and scanning happens locally on your machine.
